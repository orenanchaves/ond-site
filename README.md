# OND — Site

Site institucional do OND (landing público).

## Páginas

- `index.html` — Home ("Organize sua viagem com IA")
- `agencias.html` — OND para Agências (B2B)

## Como publicar uma alteração

Edite os arquivos e mande pro GitHub. O GitHub Pages atualiza o site no ar sozinho.

```bash
git add -A
git commit -m "atualiza site"
git push
```
